# 📡 MeraVPN — Высокопроизводительный VPN-клиент нового поколения для Android

MeraVPN — это современное высокотехнологичное Android-приложение, спроектированное для безопасного обхода блокировок и систем глубокого анализа пакетов (DPI) провайдеров. Клиент разработан на языке **Kotlin** с использованием декларативного UI фреймворка **Jetpack Compose** и полностью соответствует стандартам Google Material Design 3.

---

## 🎨 Визуальный стиль и дизайн (Material 3)

Приложение предлагает элегантный интерфейс, спроектированный с учетом удобства использования и эстетики:
- **Интуитивный Dashboard**: Весь контроль подключения сосредоточен на одном главном экране. Красивая и понятная индикация статуса работы.
- **Интерактивный глобус**: Анимированная карта с импульсирующими эффектами активности, визуализирующая статус вашего безопасного туннеля.
- **Темная и светлая космическая темы**: Поддержка динамической цветовой палитры Android и контрастного ночного режима для минимизации усталости глаз.
- **Встроенная аналитика трафика**: Графики скорости скачивания и отдачи данных в реальном времени с точной детализацией потребления.

## 🛠 Ключевые технические возможности

MeraVPN объединяет передовые концепции сетевого туннелирования:
1. **Широкий спектр протоколов**: Поддержка безопасного импорта и подключения через протоколы `vless://`, `vmess://`, `ss://` (Shadowsocks), `trojan://` и экспериментальной поддержки современных протоколов нового поколения.
2. **Продвинутый DPI-Bypass (Обход ТСПУ)**:
   - **Фрагментация Xray**: Разбиение TLS-пакетов (например, `ClientHello`) на мелкие сегменты с задержкой, что полностью лишает ТСПУ возможности идентифицировать и заблокировать сигнатуру VPN. Настраиваемые параметры длины фрагментов, задержек и стратегий разделения.
   - **Шумовое зашумление (Noise/Padding)**: Добавление случайных или преопределённых данных в пакеты для изменения их размера и энтропии, делая трафик неотличимым от обычного веб-серфинга.
3. **Маршрутизация Трафика**:
   - Переключение схем маршрутов напрямую из настроек (например, обход определенных доменных зон, туннелирование только RU-трафика или глобальный режим).
4. **Умное управление подписками**:
   - Быстрый импорт узлов списком из буфера обмена или через QR-коды.
   - Автоматическая синхронизация подписок по внешним ссылкам.
5. **Тестирование пинга и автовыбор лучшего**:
   - Ультрабыстрый параллельный тест задержки (TCP Connect Ping) для всей базы узлов.
   - Полноценная функция автоматического переключения на сервер с кратчайшим пингом.
6. **Live-Логирование и Диагностика**:
   - Встроенный интерактивный реестр системных логов в реальном времени (`LiveLoggerView`), позволяющий детально отслеживать протокол рукопожатий и быстро находить критические ошибки.
   - Возможность беспрепятственного экспорта текстовых файлов логов через стандартное меню «Поделиться».

## 👑 Ролевой Центр Управления & Messenger Поддержки

В приложение интегрирована полноценная, локально сохраняемая СУБД-система управления учетными записями, тикетами обратной связи и глобальными оповещениями:

1. **Глобальное вещание**:
   - Главный Создатель (Михаил Попов) может запускать оповещения на всё приложение. Текст моментально выводится в виде акцентного всплывающего баннера на главном экране всех пользователей.
2. **Пасхалка — Роскомнадзор**:
   - В систему встроен специальный пасхальный аккаунт `Роскомнадзор`. По умолчанию по понятным причинам он заблокирован создателем.
   - Главный Администратор (Михаил Попов) имеет абсолютные полномочия разбанить или забанить Роскомнадзор обратно в реальном времени с помощью интерактивного переключателя!
3. **Иерархия модерации чатов**:
   - **Младший Модератор**: Может просматривать общую базу тикетов поддержки пользователей и давать ответы, которые автоматически отображаются у пользователя с пометкой `[Младший модератор]`.
   - **Старший Администратор**: Контролирует всю систему модерации, отвечает на тикеты обратной связи и может блокировать/разблокировать младших модераторов.
   - **Создатель (Михаил Попов)**: Обладает полной свободой действий (Broadcasting + Бан Роскомнадзора).
4. **Защитная Математическая Капча (Anti-Spam)**:
   - Чтобы переключить аккаунт, войти в админку или зарегистрировать новую роль, пользователю необходимо решить простую динамическую капчу на сложение чисел. Это защищает систему от перегрузки и предотвращает зависание UI.

## 🏗 Архитектура приложения

Проект спроектирован с соблюдением современных архитектурных паттернов Android (MVVM и Clean Architecture):
- **UI слои**: Полноценный декларативный Compose-код без XML (`VpnScreens.kt`), использующий `StateFlow` для реактивного обновления ресурсов.
- **Живой цикл VPN-туннеля**: Реализован в фоновой службе `VpnConnectionService` (расширяет системный `VpnService`), управляющей дескрипторами туннеля (`ParcelFileDescriptor`) на уровне операционной системы.
- **Бизнес-логика**: Изолирована в `VpnViewModel.kt`. Хранит текущую сессию, параметры оптимизации и инициирует асинхронные операции на безопасных диспетчерах корутин (`Dispatchers.IO`).
- **Локальное хранилище данных**: Локальная легковесная СУБД SQLite через класс-помощник `VpnDatabaseHelper.kt`, обеспечивающий полную транзакционность сохранения профилей, учетных записей, писем поддержки и подписок.

## 🚀 Сборка проекта

Для сборки проекта на локальной машине или в CI-системах не требуется сложной предварительной настройки:

### Системные требования
- **JDK (Java Development Kit)**: Версии 17 или выше (рекомендуется Azul Zulu JDK 17).
- **Android SDK**: `compileSdk = 35`, `minSdk = 26`, `targetSdk = 35`.

### Запуск сборки из командной строки
Для локальной сборки проекта выполните команду:
```bash
# Очистка и сборка отладочного APK
gradle assembleDebug
```

Сгенерированный установочный файл `.apk` будет сохранён в следующей директории:
`app/build/outputs/apk/debug/app-debug.apk`

## 🔒 Безопасность и конфиденциальность

MeraVPN не использует внешние аналитические службы и не отправляет пользовательские данные на сторонние серверы. Все конфигурационные ключи, приватные параметры серверов и журналы логов хранятся строго локально на вашем мобильном устройстве в зашифрованной песочнице Android.
